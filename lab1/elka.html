Лабораторная работа №2

Игра на JavaScript

Задачи лабораторной работы

  • Создать веб-страницу (т.е. одну HTML страницу) с названием elka.html. Это будет страница – часть вашего веб-портала.

  • Ссылку на эту страницу добавить в меню файла index.html из лабораторной 1.

  • Приготовьте файл с картинкой минимум 4 типов елочных игрушек и картинку елки. Они могут отличаться цветом, формой и размером. Рекомендуется готовить картинки в формате png с включенным альфа каналом (вместо белого фона прозрачный).

  • В части страницы создайте блок с типами игрушек. Поверх каждой из игрушек должна находиться иконка «+», дающая пользователю понять, что нажав на кнопку, на елку добавится новая игрушка.

  • Добавьте обработчики событий на каждую игрушку, чтобы по нажатии на картинку, на страницу добавлялась одна игрушка выбранного типа, в случайном месте на экране (в видимой области).

  • Для каждой добавленной игрушки добавьте обработчик событий, отвечающий за перетаскивание игрушек курсором по странице.

  • Добавьте изображение елки на страницу.

  • Создайте обработчик событий для елки, чтобы при перетаскивании курсором (drag and drop) елка двигалась в нужном направлении. Если картинки добавленных игрушек находятся поверх елки, то они передвигаются вслед за елкой.

  • Добавьте кнопку «построить имя» на страницу рядом с кнопками добавления игрушек

  • Добавьте обработчик событий кнопки, при нажатии на которую все добавленные игрушки упорядочиваются в форме Вашего имени. Имя формируется слева направо: если на страницу добавлено мало игрушек, имя сформируется не до конца.
